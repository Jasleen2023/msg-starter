Links and underlines

[p > a] Links with default underline
clipping descending letters depends on browser.

[a] Link with default underline
(a tags are inline elements, so they don't pop in a margin-bottom, because it wouldn't do anything anyway.)
 

[p > span.underline] Pretty text underline without
clipping descending letters.

[p.underline] Pretty text underline without
clipping descending letters. (probably don't put the underline class directly on a p... unless you want only the first line to be underlined?)

[p > a.underline] a link within a p
and the underline gqy|

[a.underline] just an a with
the underline class
